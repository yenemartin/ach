# V1 Scope

## Product Position

ACH v1 is a marketing website platform for adult family homes.

It is not an operations platform, a clinical tool, or a family portal.

## Public Platform Experience

- central directory website
- searchable list of adult family homes
- filters for city or area, availability, care tags, room type tags, and language or cultural fit tags
- each listing links to a branded subdomain such as `sunrise-home.afhcares.com`

## Per-Home Public Website

Each home site should support:

- home name
- hero image or logo
- address and map block
- phone and contact CTA
- about/story section
- amenities and care highlights
- room descriptions
- availability status
- promotional content
- photo gallery
- contact-for-pricing messaging
- contact or tour request form

## Operator Admin

Minimum v1 admin workflow:

- create a home record
- assign slug and subdomain under `afhcares.com`
- upload branding and gallery images
- edit about text, amenities, room descriptions, and promotions
- manually update availability
- review contact and tour requests
- publish or unpublish a home

## Compliance Boundary

v1 must avoid:

- resident-care or medical data collection
- document upload for admissions or care workflows
- hidden draft-home exposure in public endpoints
